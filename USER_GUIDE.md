# StagePV 使用指南

StagePV 是一個專業的 **3D 舞台視覺預覽系統**，專為演唱會、活動及展覽的 LED 牆面內容預覽而設計。

---

## 功能概覽

| 功能 | 說明 |
|------|------|
| 🎮 **自由測試** | 上傳 3D 模型並在本地預覽影像/影片內容 |
| 📹 **影像進度** | 雲端專案管理，支援 R2 影片上傳與分享 |
| 🔗 **分享功能** | 生成可分享的唯讀連結 |

---

## 快速入門

### 1. 訪問系統
開啟瀏覽器，前往系統網址：
- 本地開發：`http://localhost:3000`
- 線上版本：您的部署網址

### 2. 選擇模式
首頁會顯示兩個選項：
- **自由測試** - 快速預覽，無需登入
- **影像進度** - 專案管理與雲端分享

---

## 🎮 自由測試模式

### 適用情境
- 快速測試 3D 模型渲染效果
- 預覽本地影片/圖片在 LED 牆面上的呈現
- 演示或客戶展示

### 操作步驟

#### 1. 上傳 3D 模型
點擊左側面板的 **「上傳模型」** 按鈕，選擇 `.glb` 或 `.gltf` 格式的 3D 模型檔案。

> 系統會自動壓縮模型並套用材質。

#### 2. 上傳內容
畫面左上角有一個雲朵上傳圖示，點擊可上傳：
- **圖片**：PNG、JPG
- **影片**：MP4、M4V

> ⚠️ 檔案僅在本地處理，不會上傳至雲端。

#### 3. 調整視角
- **拖曳滑鼠**：旋轉視角
- **滾輪**：縮放
- **按住滑鼠右鍵拖曳**：平移

#### 4. 切換預設視角
畫面底部的視角選擇器可快速切換：
- 正面視角
- 側面視角
- 俯視視角

#### 5. 分享場景
在管理面板點擊 **「分享」** 按鈕，系統會生成一個可分享的連結。

## 📹 影像進度模式

### 適用情境
- 多人協作的專案管理
- 追蹤影片製作進度
- 與客戶分享特定影片的預覽

### 專案管理

#### 創建專案
1. 前往影像進度頁面
2. 輸入專案名稱
3. 點擊「創建專案」

#### 上傳 R2 影片
1. 選擇一個專案
2. 在 **「R2 影片管理」** 區域點擊上傳
3. 選擇影片檔案（支援中文檔名）
4. 等待上傳完成

#### 分享單一影片
每個上傳的影片都有獨立的分享連結：
1. 點擊影片卡片上的分享圖示
2. 複製生成的連結
3. 分享給客戶或團隊成員

> 分享連結為**唯讀模式**，接收者無法編輯專案。

## 🎛️ 管理面板

管理面板位於畫面右側，提供以下功能區：

| 區域 | 功能 |
|------|------|
| 📁 **模型** | 上傳和管理 3D 模型 |
| 🖼️ **內容** | 管理圖片和影片 |
| 💡 **燈光** | 調整環境光和方向光 |
| 🪞 **反射** | 啟用完美渲染模式 |
| 🎬 **Cue** | 管理場景預設位置 |
| 📹 **R2 影片** | 管理雲端影片（僅影像進度模式） |

### 完美渲染模式
啟用後可獲得更逼真的效果：
- 地面反射
- 增強 Bloom 效果
- 更高的渲染品質

> ⚠️ 此模式較消耗資源，可能影響效能。

## 常見問題

### Q: 模型上傳後看不到？
A: 請確認：
1. 模型格式為 `.glb` 或 `.gltf`
2. 模型尺寸合理（可能需要縮放）
3. 檢查瀏覽器控制台是否有錯誤

### Q: 影片播放卡頓？
A: 建議：
1. 使用 MP4 格式（H.264 編碼）
2. 降低影片解析度
3. 關閉完美渲染模式

### Q: 分享連結無法開啟？
A: 請確認：
1. 專案已成功保存
2. 連結完整無缺漏
3. 接收者有網路連線

### Q: 如何刪除已上傳的內容？
A: 將滑鼠移至項目上方，點擊右上角的 ❌ 按鈕（部分項目需要長按）。

## 技術規格

### 支援的檔案格式
| 類型 | 格式 |
|------|------|
| 3D 模型 | `.glb`, `.gltf` |
| 圖片 | `.png`, `.jpg`, `.jpeg` |
| 影片 | `.mp4`, `.m4v` |
| 串流 | `.m3u8` (HLS) |

### 建議瀏覽器
- Chrome（推薦）
- Firefox
- Edge
- Safari

### 硬體建議
- 獨立顯示卡（啟用完美渲染模式時）
- 8GB+ RAM
- 穩定的網路連線
